# DetectAgent

DetectAgent是一个基于大语言模型的智能体，专门用于探测Java代码中的遗漏异常处理并提供修改建议。

## 使用方法

### 创建环境

```bash
cd DetectAgent
conda create --name DetectAgent python=3.10
conda activate DetectAgent
```

### 安装依赖

```bash
pip install -r requirements.txt
```

### 开启服务

```bash
python start.py
```

### 使用体验

打开html文件即可体验。
